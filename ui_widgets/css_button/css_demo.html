CSS Cheatsheet

  • CSS Selectors
    • All Elements Selector: *
    • Class Selector: .class
    • ID Selector: #id
    • Elements Selector
      • div: selects div
      • div p: selects all p in div
      • div, p: selects both div and p
      • div>p: selects p whose parent is div
      • div+p: selects p who lives immediately after div
    • Attributes Selector
  • CSS Positioner
  • CSS Box Model
    • Border
      • border-style: solid, double, dotted, dashed, groove, ridge, inset, outset
      • individual border styles: border-top-style, border-right-style, border-bottom-style, border-left-style
      • border-color
      • border-width
      • shorthand: border: border-width border-style border-color
    • Padding
    • Margin
      • individual margins: margin-top, margin-right, margin-bottom, margin-left
      • possible values: auto, px, %, inherit
